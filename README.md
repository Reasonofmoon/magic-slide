# Studio Gen V4.1

AI 기반 엔터프라이즈 슬라이드 스튜디오입니다. 마크다운을 기반으로 테마 통일 렌더링, 개별 슬라이드 편집, PDF/PPTX 내보내기를 제공합니다.

## 주요 기능
- 테마 엔진(Modern / Dark / Minimal)
- 1600px 와이드 레이아웃 + 진행 상태 시각화
- PDF/PPTX 고해상도 익스포트
- 슬라이드 개별 편집 모드

## 시작하기
```bash
npm install
npm run dev
```

### 환경 변수
- `VITE_GEMINI_API_KEY`: Gemini API Key (옵션)

## 빌드
```bash
npm run build
```

빌드 결과물은 `dist` 폴더에 생성됩니다.
